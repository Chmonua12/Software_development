## Обзор

ArtConnect — это Telegram-бот для художников, который объединяет участников арт-сообщества через портфолио, систему оценок и переходы во внешние соцсети.

**Ключевые отличия от классического дейтинг-бота:**

- Нет мэтчей и внутренней переписки
- Лайк — это оценка работы, а не сигнал к знакомству
- Акцент на переходах во внешние соцсети (Telegram, Instagram, VK, Behance)
- Публичный топ художников
- Приоритет Telegram-каналам среди соцсетей

**Рейтинговая система** строит порядок показа анкет по трёхуровневой модели:

* **Уровень 1 — первичный рейтинг**
  Оценивает качество анкеты: наличие аватарки, количество фото работ, полноту описания, возраст и город, интересы (теги), бонус за Telegram как основную соцсеть.
* **Уровень 2 — поведенческий рейтинг**
  Учитывает взаимодействие: количество лайков, соотношение лайков к просмотрам, переходы по ссылкам на соцсети, временную активность (в какое время чаще лайкают).
* **Уровень 3 — комбинированный рейтинг**
  Суммирует первые два уровня с весами (40% и 60%) + бонусы за рефералов.

**Техническая архитектура:**

- aiogram (бот) + FastAPI (API-шлюз)
- PostgreSQL + Minio (S3) + Redis
- Kafka + Celery (очереди и воркеры)
- Prometheus + Loki (мониторинг)

## Ключевые пользовательские сценарии

* **Старт и регистрация**
  Пользователь запускает бота командой `/start`. Бот привязывает Telegram-аккаунт к записи в базе и создаёт заготовку профиля.
* **Заполнение анкеты**
  Бот собирает данные: имя, возраст, город, описание творчества, аватарку и до 5 примеров работ. Фото загружаются в Minio (S3). Затем — настройка соцсетей: выбор платформы (Telegram, Instagram, VK, Behance), ввод и валидация ссылки. Telegram имеет приоритет как основная соцсеть.
* **Лента художников**
  Бот показывает анкеты в порядке комбинированного рейтинга. Карточка содержит: аватарку, имя, возраст, город, описание, примеры работ (с листанием), кнопки соцсетей. Внизу — кнопки «Лайк для общения», «Лайк в избранное» и «Следующий».
* **Лайк и топ**
  Реализовано **два типа лайков**:
  - **Лайк для общения** — сигнал о заинтересованности в контакте. При взаимном лайке для общения образуется **мэтч**, после чего пользователи могут начать переписку внутри бота.
  - **Лайк в избранное** — добавляет анкету в личный список «избранного» без создания мэтча. Это оценка работы или желание сохранить анкету для последующего просмотра.
  
  Лайки увеличивают счётчик художника и влияют на рейтинг. Команда `/top` показывает топ-10 по лайкам с кнопками соцсетей. Топ кэшируется в Redis и обновляется каждые 5 минут.
* **Реферальная система**
  У пользователя есть реферальная ссылка — за каждого приглашённого друга он получает бонус к комбинированному рейтингу.
